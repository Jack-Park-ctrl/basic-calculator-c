# 🧠 Basic Calculator in C

C 언어로 구현한 콘솔 기반 계산기 시뮬레이터입니다.  
사용자는 메뉴를 통해 연산을 선택하고 숫자를 입력하여 실시간으로 결과를 출력받을 수 있습니다.

## ✅ 지원 기능

- 덧셈
- 뺄셈
- 곱셈
- 나눗셈
- 나머지
- 제곱 연산

## 🛠 파일 구성

| 파일명 | 설명 |
|--------|------|
| `main.c` | 메인 함수 및 사용자 입력 처리 |
| `calculator.c` | 연산 함수 정의 |
| `calculator.h` | 함수 선언 (헤더) |
